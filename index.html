Introduction to Ubuntu

What is Ubuntu Linux and why should you use it?

Ubuntu Linux is a free to use operating system that competes with other for-profit OSs such as Apple's Mac-OS and Microsoft's Windows platforms. Linux exists as a host of various distributions or flavors that are all open source, meaning that anyone can get a copy of the source code and contribute to updates or to software compatibility.

There are many distributions in various forms, and many are highly specialized for specific tasks such as computer development, hacking, or general-use daily drivers. Ubuntu is a general-use distribution that strives for the same level of accessibility and ease of use as other mainstream platforms such as MacOS. It is one of the most popular and widely used Linux distributions, so naturally there is a large number of developers contributing to tasks and applications from writing hardware drivers all the way to word processors and web browsers.

Because of this, Ubuntu is highly regarded as the most beginner friendly Linux distributions. Ubuntu also has a professional development team that stands behind the source code of the core framework of the operating system. This means that Ubuntu Linux is also one of the most stable distributions. With applications widely available and a graphical interface that is intuitive and thoughtful, Ubuntu is one of the best free options for a replacement operating system.

Computer Requirements for Ubuntu
  • 2GHz Dual Core Processor or Better
  • At Least 4GB RAM
  • At Least 25GB Hard Drive
  • USB Port
  • Internet Access*

*You may need access to ethernet cable for wired internet connection, or a usb cable to use a mobile phone as a mobile hot-spot to set up Wi-Fi drivers.

You Will Need
  • USB Drive 2GB+
  • ISO utility capable of creative Live Boot USB (I.E. Rufus)
  • Ubuntu ISO Download for Current version LTS (Long Term Support)

Download and Setup Ubuntu Live Boot USB

In this first step, you will navigate to Ubuntu's website and download the Ubuntu ISO


Navigate to ubuntu.com

On the navigation bar, click Download

Under Ubuntu Desktop click on the green box with LTS and the version number


In the next step, you will use the Ubuntu ISO file to create a Live Boot USB.

!WARNING!

THE NEXT STEPS WILL FORMAT YOUR USB DRIVE.

ALL DATA ON THE USB DRIVE BEFORE WRITING THE ISO TO THE DRIVE WILL BE ERASED!

BACK UP ANY IMPORTANT DATA BEFORE PROCEEDING!


Insert your USB drive and run your ISO utility.

Mount the Ubuntu ISO and create a Live Boot on your USB drive.

Rufus Setup

Once you have opened Rufus, on the Drive Properties window:

Device should be set to your USB Drive

-If you are unsure which is your USB Drive, open a windows explorer window, navigate to My Computer, and take note of which letter is assigned to your USB Drive

Set Boot Selection to Disk or ISO image

Click on the SELECT button next to the Boot Selection option and navigate to the downloaded ISO file of Ubuntu, and select it.

Set Partition Scheme to MBR

Set Target System to BIOS or UEFI

Format Options should be left alone, but as long as your USB drive is selected properly it should show File System FAT32 and Cluster Size should show the size of the USB drive (IE 1024MB is equivalent to 1GB)

Click Start to begin the process. If there is a pop-up window part-way through the process, select the (Recommended) option and click OK to continue

This writing process could take up to 5 or 10 minutes depending on the computer.

Installing Ubuntu Linux

In this next step, you will restart your computer, and set your BIOS to boot into the Live Boot USB.

Restart your computer. When the logo splash screen appears, press the BIOS key. This will vary between manufacturers, but for most computers this will be F2 or F10.
The startup splash screen will specify.

Once in BIOS, depending on your BIOS manufacturer, change the order of Boot Drives, or select which Drive you wish to boot from and press enter to proceed.

Once you have set your Boot Drive Order, your computer should now boot into a simple screen with some options to choose from.

Select “Try Ubuntu” to boot into a Live version. It is recommended to boot into Ubuntu first, as this will allow the user to make sure the operating system runs properly, and connect to the internet, making the following steps much easier.

Once loaded into Ubuntu, on the top right corner click on the down arrow and then the wifi symbol. If Ubuntu's default drivers are compatible with your computer's hardware, you should see available WiFi networks. Otherwise you may need to access the internet via Ethernet, or a mobile WiFi hotspot.

NOTE: Ubuntu comes preinstalled with some basic applications. To make sure internet and audio drivers are functioning as intended, play a video on youtube.

Close any open windows, and on the desktop double click “Install Ubuntu.”

During the installation procedure, set your preferred language and keyboard layout.

Make sure the “Download Updates While Installing Ubuntu” and “Install third-party software for graphics and wi-fi hardware and additional media formats” boxes are checked, and then select “Normal Installation.”

On the “Installation Type” screen, select “Erase disk and install Ubuntu”

Following this step, set your Time Zone and login credentials.

The remaining process could take approximately 10 minutes or less. After the process is finished, select “restart now.” Follow the prompts and after the computer is finished restarting, log in using the credentials set previously.

You can choose to follow through the first time setup at startup or skip through and complete these steps later.

First Time Setup

“Connect your online accounts.” You can choose to log in to your existing Google, Microsoft, etc accounts in this step, or skip this step by clicking “Skip Step” at the top right of the window.

Livepatch setup will install security updates automatically, and is a free service for personal accounts. It is recommended to activate Livepatch service, or you can skip this step by clicking “Skip Step” at the top right of the window.

“Help Improve Ubuntu?” You can choose either option at your preference. Agreeing to send data will aid the developers in use-case metrics, crash reports, bug reports, etc. To see what kind of data these reports collect and report, click on “Show first report.”

On the Privacy window you can choose whether to activate location services. This simply allows the computer and applications to access your geolocation data. This will aid applications and web browsers to provide accurate service by knowing your location. This step is user preference.

The “Ready to go” screen shows an example range of free software applications you can install by opening the software application menu. Click “Done” in the top right, or “Open Software Now” in the bottom left to proceed.
For future use, the Ubuntu Software application is located on the task bar on the left side of the screen (Orange Suitcase).

Congratulations, you have successfully installed Ubuntu Linux!

Feel free to explore Software applications (Orange Suitcase) or get customizing with the settings menu. To pull up all applications, click on the arrangement of squares at the end of the main task bar, located bottom left of the screen by default.
